# Real Time Shapes detection
This script detects shapes present in an image. 
The user can choose the minimum area to detect as well 
as the thresholds.

### Example
![Example](Example.gif)

### Running the script
```commandline
python detector.py
```

### Libraries Used
- OpenCV
- Numpy
